# Alperen's Kebab

A restaurant landing page created as part of **The Odin Project** JavaScript curriculum.  
The project focuses on building a restaurant website with **Webpack**, **JavaScript modules**, and **DOM manipulation**.

## Features

- Home page with restaurant introduction
- Menu page with kebab dishes
- About page with address, opening hours and legal information
- Tab-based navigation
- Content generated dynamically with JavaScript
